# 腾讯云部署工具

## 简介

这是一个用于自动化部署 WEBconfig 项目到腾讯云的工具，封装了完整的构建和部署流程。

## 功能特点

- ✅ 自动检查 tcb 命令行工具安装状态
- ✅ 自动执行项目构建
- ✅ 检查构建产物完整性
- ✅ 一键部署到腾讯云
- ✅ 显示详细的部署结果

## 前置条件

### 1. 安装 tcb 命令行工具

```bash
npm install -g @cloudbase/cli
```

### 2. 登录腾讯云

```bash
tcb login
```

按照提示完成登录操作，确保您有权限访问目标环境。

### 3. 项目配置

确保项目中包含以下文件：
- `package.json` - 项目依赖配置
- `vite.config.js` - 构建配置

## 使用方法

### 方法一：直接执行脚本

```bash
# 进入技能目录
cd skills/tencent-cloud-deploy

# 赋予执行权限
chmod +x deploy.sh

# 执行部署
./deploy.sh
```

### 方法二：从项目根目录执行

```bash
# 执行部署
./skills/tencent-cloud-deploy/deploy.sh
```

## 部署配置

- **部署路径**：`/MahStudio`
- **腾讯云环境**：`dev-1gjaefwhe1e8a952`
- **构建产物目录**：`dist`

## 部署流程

1. **检查环境** - 验证 tcb 命令行工具是否安装
2. **构建项目** - 执行 `npm run build`
3. **验证产物** - 检查构建结果是否完整
4. **部署上传** - 上传构建产物到腾讯云
5. **结果反馈** - 显示部署成功或失败信息

## 常见问题

### Q: 执行脚本时提示 "Permission denied"

**A:** 请先赋予脚本执行权限：
```bash
chmod +x ./skills/tencent-cloud-deploy/deploy.sh
```

### Q: 部署失败，提示 "未找到环境"

**A:** 请确保您已登录腾讯云，并且有权限访问 `dev-1gjaefwhe1e8a952` 环境。

### Q: 构建失败

**A:** 请检查项目依赖是否完整，执行 `npm install` 后重试。

## 注意事项

- 部署前请确保代码已提交，避免部署未完成的代码
- 部署过程中请保持网络连接稳定
- 部署完成后请访问域名验证部署结果
