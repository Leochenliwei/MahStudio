# 实现属性面板元件组的网格排版

## 目标
为属性面板中的单选和复选框组实现网格排版，支持一行多个选项，并根据文字长度自动换行。

## 实现步骤

### 1. 修改 Workbench.vue 中的样式
- 修改 `.checkbox-group` 样式，将 `flex-direction: column` 改为 `flex-wrap: wrap`，并添加适当的间距
- 修改 `.toggle-group` 样式，同样将 `flex-direction: column` 改为 `flex-wrap: wrap`，并添加适当的间距
- 为 `.checkbox-option` 和 `.toggle-option` 添加 `flex-basis` 或 `min-width` 以控制每行显示的选项数量
- 确保选项文字过长时能够自动换行

### 2. 保持 AdvancedRulesPage.vue 的现有实现
- AdvancedRulesPage.vue 中的复选框组已经使用了 `flex-wrap: wrap`，可以参考其实现方式
- 确保两处实现保持一致的样式

### 3. 测试和优化
- 测试不同长度的选项文字，确保自动换行正常工作
- 测试不同数量的选项，确保网格排版效果良好
- 确保在不同屏幕尺寸下的响应式表现

## 技术要点
- 使用 CSS Flexbox 实现灵活的网格布局
- 通过 `flex-wrap: wrap` 实现自动换行
- 通过调整间距和最小宽度确保布局美观
- 确保文字换行时的对齐和美观

## 预期效果
- 复选框和单选框组将以网格形式排列，一行显示多个选项
- 选项文字过长时会自动换行，保持布局整洁
- 在不同屏幕尺寸下都能保持良好的显示效果
