# 修改项目结构为管理台和工作台两个页面

## 1. 项目分析

当前项目是一个单页应用，所有功能都在 App.vue 中实现：
- 左侧是后台管理菜单
- 右侧根据选择显示不同的内容，包括游戏配置列表和工作台

用户需求是将其改为两个独立的页面：
- 管理台页面：包含游戏配置列表
- 工作台页面：包含可拖拽面板
- 点击游戏配置列表中的任意游戏，会跳转到工作台页面

## 2. 技术方案

- **路由管理**：安装并配置 Vue Router
- **页面结构**：创建两个主要页面组件
- **状态管理**：使用 Vue 3 Composition API + provide/inject 或 Pinia 进行跨页面数据传递
- **拖拽面板**：继续使用 vue-draggable-resizable

### 3.4 路由配置

```javascript
// src/router/index.js
import { createRouter, createWebHistory } from 'vue-router'
import Admin from '../views/Admin.vue'
import Workbench from '../views/Workbench.vue'

const routes = [
  {
    path: '/',
    name: 'Admin',
    component: Admin
  },
  {
    path: '/workbench',
    name: 'Workbench',
    component: Workbench,
    props: true
  }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router
```
