# 实现画布区域的拖拽、滚动和缩放功能

## 1. 分析当前画布实现
- 画布区域使用 `.workbench-canvas` 类的 div 元素
- 目前已经实现了基本的组件拖拽和放置功能
- 需要添加视图级别的拖拽、滚动和缩放功能

## 2. 实现键盘空格拖拽视图
- 添加空格键按下/释放事件监听器
- 当空格键按下时，启用拖拽模式
- 监听鼠标移动事件，计算位移并更新画布位置
- 当空格键释放时，禁用拖拽模式

## 3. 实现鼠标中键拖拽视图
- 添加鼠标中键按下/释放事件监听器
- 当鼠标中键按下时，启用拖拽模式
- 监听鼠标移动事件，计算位移并更新画布位置
- 当鼠标中键释放时，禁用拖拽模式

## 4. 支持鼠标滚轮上下滑动视图
- 添加鼠标滚轮事件监听器
- 计算滚轮滚动量并转换为画布的垂直位移
- 更新画布位置以实现滚动效果

## 5. 支持视图缩放功能
- 添加鼠标滚轮事件监听器，同时检测修饰键
- 对于 Windows：检测 alt+滚轮组合
- 对于 Mac：检测 command+滚轮组合
- 根据滚轮方向计算缩放因子
- 更新画布的缩放比例和位置

## 6. 实现画布位置和缩放的状态管理
- 添加画布位置（x, y）和缩放比例的响应式变量
- 在画布元素上应用这些变量作为变换样式
- 确保拖拽和缩放操作正确更新这些变量

## 7. 测试和优化
- 测试所有实现的功能是否正常工作
- 优化拖拽和缩放的性能
- 确保在不同设备和浏览器上的兼容性

## 8. 代码实现步骤
1. 在 `<script setup>` 部分添加画布状态变量
2. 在 `<template>` 部分修改画布元素，添加变换样式
3. 实现事件监听器和处理函数
4. 添加必要的辅助函数
5. 测试所有功能

## 9. 技术要点
- 使用 Vue 的响应式系统管理画布状态
- 使用 CSS transform 属性实现视图的平移和缩放
- 正确处理事件冒泡和默认行为
- 优化事件处理以提高性能
- 确保跨平台兼容性
