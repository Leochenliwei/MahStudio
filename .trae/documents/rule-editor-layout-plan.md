# RuleEditorModal 布局调整计划

## 目标
将 RuleEditorModal.vue 的界面调整为左右分栏布局，参考截图中的样式：
- 左侧：条件判断 (IF) 区域
- 右侧：执行动作 (THEN) 区域
- 顶部：规则说明输入框
- 底部：返回和保存按钮

## 当前问题
1. 目前是上下布局，需要改为左右布局
2. 执行动作区域的样式需要调整以匹配截图

## 实施步骤

### Step 1: 调整模板结构
- 将 modal-body 改为左右分栏布局（flex 布局）
- 左侧条件区域宽度约 50%，右侧动作区域宽度约 50%
- 规则说明保持在顶部

### Step 2: 调整条件判断区域样式
- 条件行样式调整：左侧显示序号圆点
- 条件字段：组件选择下拉框、操作符下拉框、值输入框
- 添加条件和条件组按钮样式调整

### Step 3: 调整执行动作区域样式
- 参考截图中的动作卡片样式
- 动作类型使用标签页形式展示
- 满足条件时/否则配置样式调整

### Step 4: 调整样式
- 左右分栏的边框分隔线
- 各区域的 padding 和 margin
- 响应式布局（移动端改为上下布局）

## 具体变更

### 模板变更
1. modal-body 添加 `split-layout` class，使用 flex 布局
2. 条件区域和动作区域分别添加 `left-panel` 和 `right-panel` class
3. 动作区域改为卡片式布局
4. 动作类型改为标签页切换形式

### 样式变更
1. 添加 `.split-layout` 样式：display: flex, flex-direction: row
2. 添加 `.left-panel` 和 `.right-panel` 样式
3. 调整 `.action-row` 为卡片样式
4. 调整 `.action-type-tabs` 样式
5. 添加响应式样式，移动端改为上下布局

## 预期效果
- 界面呈现左右分栏布局
- 左侧条件区域和右侧动作区域清晰分隔
- 整体样式与截图一致
