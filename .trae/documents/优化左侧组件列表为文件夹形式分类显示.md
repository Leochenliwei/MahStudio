# 优化左侧组件列表为文件夹形式分类显示

## 一、优化目标

将左侧组件列表从平铺显示改为以文件夹形式按分类组织，实现以下功能：

1. **分类文件夹结构**：每个分类显示为一个可展开/折叠的文件夹
2. **展开/折叠功能**：点击文件夹可展开或折叠其中的组件
3. **保持搜索功能**：搜索关键词时，仍然可以过滤所有组件
4. **保持组件管理**：每个组件的启用/禁用开关功能保持不变
5. **视觉优化**：文件夹样式美观，交互流畅

## 二、实现方案

### 1. 修改模板结构

- 在 `<div class="components-list">` 内部，替换当前的组件列表为分类文件夹结构
- 为每个分类创建一个文件夹容器，包含分类标题和展开/折叠按钮
- 在每个分类文件夹内显示该分类下的组件

### 2. 添加状态管理

- 添加 `expandedCategories` 响应式变量，用于存储当前展开的分类
- 实现 `toggleCategory` 函数，用于切换分类的展开/折叠状态

### 3. 修改计算属性

- 修改 `filteredComponents` 计算属性，改为按分类分组返回数据
- 保持搜索和分类过滤逻辑不变

### 4. 更新样式

- 添加文件夹样式，包括折叠/展开图标
- 添加文件夹展开/折叠的动画效果
- 优化分类标题和组件列表的间距和布局

### 5. 具体修改点

1. **模板部分**：
   - 在 `.components-list` 中添加分类文件夹结构
   - 为每个分类添加展开/折叠按钮
   - 在分类文件夹内显示组件列表

2. **脚本部分**：
   - 添加 `expandedCategories` 响应式变量
   - 添加 `toggleCategory` 函数
   - 修改 `filteredComponents` 计算属性，按分类分组
   - 确保搜索和过滤功能正常工作

3. **样式部分**：
   - 添加 `.category-folder` 相关样式
   - 添加 `.folder-header` 和 `.folder-content` 样式
   - 添加展开/折叠图标的样式和动画

## 三、预期效果

- 左侧组件列表显示为分类文件夹结构
- 点击文件夹可展开/折叠查看组件
- 搜索功能仍然可以过滤所有组件
- 组件的启用/禁用功能保持不变
- 界面更加整洁，易于导航

## 四、技术要点

- 使用 Vue 3 的 `ref` 和 `computed` 实现状态管理
- 使用 CSS 实现文件夹的视觉效果和动画
- 保持现有的搜索和过滤逻辑
- 确保响应式设计，适配不同屏幕尺寸
