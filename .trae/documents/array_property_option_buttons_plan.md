# 数组类型属性选项按钮实现计划

## 任务描述
为ComponentSelector.vue文件中数组类型属性（如checkbox、multiselect）的每个选项增加一个额外的操作按钮“添加/移除”，用于手动控制选项的添加和移除。

## 分解任务

### [x] 任务1：修改checkbox类型属性的选项结构
- **优先级**：P0
- **依赖**：无
- **描述**：在checkbox类型的选项中添加“添加/移除”按钮
- **成功标准**：每个checkbox选项旁边都有一个操作按钮，点击可以添加或移除选项
- **测试要求**：
  - `programmatic` TR-1.1：点击按钮能正确修改property.defaultValue数组
  - `human-judgement` TR-1.2：按钮样式与现有UI一致，交互流畅
- **实现细节**：在checkbox-option元素中添加按钮，根据选项是否在defaultValue数组中显示“移除”或“添加”文本

### [x] 任务2：添加按钮点击事件处理函数
- **优先级**：P0
- **依赖**：任务1
- **描述**：实现处理按钮点击事件的函数，用于添加或移除选项
- **成功标准**：点击按钮能正确更新property.defaultValue数组，并触发updateComponentProperty事件
- **测试要求**：
  - `programmatic` TR-2.1：点击“添加”按钮能将选项值添加到数组中
  - `programmatic` TR-2.2：点击“移除”按钮能将选项值从数组中移除
- **实现细节**：创建toggleArrayOption函数，处理数组的添加和移除操作

### [x] 任务3：优化按钮样式和布局
- **优先级**：P1
- **依赖**：任务1
- **描述**：为新添加的按钮添加合适的样式，确保与现有UI风格一致
- **成功标准**：按钮样式美观，与现有控件风格统一，布局合理
- **测试要求**：
  - `human-judgement` TR-3.1：按钮样式与现有UI一致
  - `human-judgement` TR-3.2：按钮布局合理，不影响现有功能
- **实现细节**：添加按钮样式，调整checkbox-option的布局结构

### [x] 任务4：测试功能完整性
- **优先级**：P1
- **依赖**：任务1、2、3
- **描述**：测试所有功能是否正常工作
- **成功标准**：所有数组类型属性的选项按钮都能正常工作
- **测试要求**：
  - `programmatic` TR-4.1：所有数组类型属性的按钮功能正常
  - `human-judgement` TR-4.2：整体UI效果良好，用户体验流畅
- **实现细节**：测试不同场景下的按钮功能

## 技术实现要点
1. 在checkbox-group的每个option中添加按钮元素
2. 根据选项是否在defaultValue数组中动态显示按钮文本
3. 实现toggleArrayOption函数处理按钮点击事件
4. 确保按钮状态与checkbox状态同步
5. 添加合适的按钮样式

## 预期效果
- 每个checkbox选项旁边都有一个“添加/移除”按钮
- 点击按钮可以手动添加或移除选项
- 按钮状态与checkbox状态保持同步
- 整体UI风格一致，用户体验良好
