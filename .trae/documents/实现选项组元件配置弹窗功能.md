# 实现选项组元件配置弹窗功能

## 需求分析
当用户在显示分组中添加一个选项组元件"单选/复选"时，右侧应该弹出一个选项组配置弹窗，允许用户配置选项组的属性和选项内容。

## 实现方案

### 1. 扩展Workbench.vue
- 添加选项组组件的状态管理变量
- 实现添加选项组组件的逻辑
- 集成Drawer组件，实现右侧弹窗功能
- 添加选项组配置相关的方法

### 2. 利用现有Drawer组件
- 复用现有的右侧抽屉功能
- 确保Drawer组件支持选项组配置
- 传递必要的props和事件处理

### 3. 实现核心功能
- 添加选项组组件时触发右侧弹窗
- 支持配置选项组名称、类型（单选/复选）
- 支持添加、编辑、删除选项
- 支持设置默认选中选项
- 保存配置后更新组件状态

### 4. 技术细节
- 使用Vue 3的Composition API
- 利用现有的状态管理和事件传递机制
- 保持与项目现有代码风格一致
- 遵循UI规范和交互设计

## 实现步骤

1. **修改Workbench.vue**
   - 添加选项组相关的状态变量
   - 实现添加选项组组件的方法
   - 集成Drawer组件并配置右侧抽屉
   - 添加事件处理方法

2. **确保Drawer组件兼容性**
   - 验证右侧抽屉功能是否完整
   - 确保选项配置界面可用

3. **测试功能**
   - 测试添加选项组组件时是否弹出配置弹窗
   - 测试选项组配置功能是否正常
   - 测试保存配置后组件状态是否更新

## 预期效果
当用户添加"单选/复选"选项组元件时，右侧会弹出一个配置弹窗，用户可以：
- 设置选项组名称
- 配置选项内容（添加、编辑、删除选项）
- 设置默认选中的选项
- 保存配置后，组件会显示在对应分类中

此实现将增强用户体验，使选项组配置更加直观和高效。
