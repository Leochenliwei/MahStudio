# 实现游戏列表页面新增游戏功能

## 1. 在游戏列表页面添加新增游戏按钮
- 在 `Admin.vue` 文件的游戏配置列表部分（表格上方）添加一个新增游戏按钮
- 按钮样式应与现有按钮保持一致，使用主色调

## 2. 创建新增游戏弹窗
- 在 `Admin.vue` 中添加弹窗组件，包含以下元素：
  - 弹窗标题："新增游戏"
  - 游戏名输入框
  - 确认和取消按钮
- 弹窗样式应遵循项目的设计规范

## 3. 实现新增游戏逻辑
- 在 `Admin.vue` 的 `<script setup>` 中添加以下功能：
  - 弹窗显示/隐藏状态管理
  - 游戏名输入值管理
  - 新增游戏方法：
    - 生成新的唯一ID（基于现有最大ID+1）
    - 创建新游戏配置对象
    - 将新游戏添加到 `gameConfigs` 数组
    - 关闭弹窗并清空输入

## 4. 实现确认后进入工作台功能
- 修改新增游戏的确认逻辑，在添加新游戏后：
  - 获取新创建游戏的ID
  - 调用 `enterWorkbench` 方法进入工作台
  - 传递游戏名参数以便在工作台中显示

## 5. 测试功能
- 验证新增游戏按钮是否显示正常
- 验证弹窗是否能正常打开和关闭
- 验证输入游戏名后是否能正确添加到列表
- 验证确认后是否能直接进入工作台
- 验证新游戏在列表中的显示是否正确

## 技术实现要点
- 使用 Vue 3 的 Composition API 进行状态管理
- 遵循项目的代码风格和设计规范
- 确保新增功能与现有功能的一致性
- 提供清晰的用户反馈和错误处理
