# 在工作台底部添加底部悬浮工具bar

## 一、实现目标

在工作台底部添加一个底部悬浮工具bar，包含以下入口：
- 模板库
- 变量管理
- 创房编辑器

参考截图2的设计风格，相关跳转逻辑使用暂未开发的提示语。

## 二、实现方案

### 1. 修改模板结构

在 `workbench-main` 容器内，在 `workbench-canvas` 下方添加一个 `bottom-toolbar` 组件：

```vue
<!-- 底部悬浮工具栏 -->
<div class="bottom-toolbar">
  <div class="toolbar-content">
    <button class="toolbar-item" @click="openTemplateLibrary">
      <Icon name="layout" size="18" />
      <span>模板库</span>
    </button>
    <button class="toolbar-item" @click="openVariableManagement">
      <Icon name="sliders" size="18" />
      <span>变量管理</span>
    </button>
    <button class="toolbar-item" @click="openRoomCreator">
      <Icon name="edit" size="18" />
      <span>创房编辑器</span>
    </button>
  </div>
</div>
```

### 2. 添加脚本逻辑

在脚本部分添加三个点击事件处理函数：

```javascript
// 打开模板库
function openTemplateLibrary() {
  alert('模板库功能暂未开发')
}

// 打开变量管理
function openVariableManagement() {
  alert('变量管理功能暂未开发')
}

// 打开创房编辑器
function openRoomCreator() {
  alert('创房编辑器功能暂未开发')
}
```

### 3. 添加样式

在样式部分添加底部悬浮工具栏的样式：

```css
/* 底部悬浮工具栏 */
.bottom-toolbar {
  position: absolute;
  bottom: 20px;
  left: 50%;
  transform: translateX(-50%);
  background-color: var(--color-surface);
  border: 1px solid var(--color-border);
  border-radius: var(--border-radius-lg);
  box-shadow: var(--shadow-md);
  padding: var(--spacing-3);
  z-index: 100;
}

.toolbar-content {
  display: flex;
  gap: var(--spacing-4);
}

.toolbar-item {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: var(--spacing-1);
  padding: var(--spacing-3);
  background: none;
  border: none;
  color: var(--color-text-primary);
  cursor: pointer;
  transition: all var(--transition-normal);
  border-radius: var(--border-radius-md);
}

.toolbar-item:hover {
  background-color: var(--color-surface-hover);
  color: var(--color-primary);
  transform: translateY(-2px);
}

.toolbar-item Icon {
  font-size: 18px;
}

.toolbar-item span {
  font-size: var(--font-size-xs);
  font-weight: var(--font-weight-medium);
}
```

## 三、预期效果

- 底部悬浮工具bar固定在工作台底部中央
- 包含三个入口：模板库、变量管理、创房编辑器
- 每个入口包含图标和文字
- 鼠标悬停时有动画效果
- 点击时显示"暂未开发"的提示语

## 四、技术要点

- 使用绝对定位实现底部悬浮效果
- 使用flex布局排列工具按钮
- 添加过渡动画提升用户体验
- 使用alert函数显示暂未开发的提示语
