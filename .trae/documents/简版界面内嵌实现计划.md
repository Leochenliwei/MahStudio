# 简版界面内嵌实现计划

## 目标
将简版选项联动界面从独立页面改为在 RoomCreatorPage 内部显示，保持 Tab 栏选中状态，不再跳转到新页面。

## 当前状态
- 简版按钮点击后通过 `goToSimpleDependency()` 跳转到 `/workbench/:id/simple-dependency` 路由
- 这是一个独立的页面，Tab 栏状态丢失

## 修改方案

### 1. 修改 Tab 切换逻辑
**文件**: `src/views/RoomCreatorPage.vue`

**修改点**:
- 将简版按钮改为 Tab 切换模式，使用 `activeTab` 控制
- 添加新的 Tab 值 `'simple-dependency'`
- 按钮添加 `:class="{ active: activeTab === 'simple-dependency' }"`
- 点击事件改为 `@click="activeTab = 'simple-dependency'"`

### 2. 添加简版内容区域
**文件**: `src/views/RoomCreatorPage.vue`

**修改点**:
- 在主内容区添加简版界面的渲染条件
- 当 `activeTab === 'simple-dependency'` 时显示简版内容
- 将 `SimpleDependencyPage` 作为组件导入并嵌入

### 3. 调整组件导入
**文件**: `src/views/RoomCreatorPage.vue`

**修改点**:
- 导入 `SimpleDependencyPage` 组件
- 删除 `goToSimpleDependency` 方法（不再需要）

### 4. 路由调整（可选）
**文件**: `src/router/index.js`

**修改点**:
- 可以选择保留或删除 `/workbench/:id/simple-dependency` 路由
- 如果保留，可以重定向到主页面并设置对应 Tab

## 具体代码变更

### RoomCreatorPage.vue 模板部分

```vue
<!-- Tab 按钮修改 -->
<button
  class="tab-item"
  :class="{ active: activeTab === 'simple-dependency' }"
  @click="activeTab = 'simple-dependency'"
>
  <Icon name="table" size="16" />
  选项联动（简版P0）
</button>

<!-- 主内容区添加 -->
<div v-if="activeTab === 'simple-dependency'" class="simple-dependency-content">
  <SimpleDependencyPage :embedded="true" />
</div>
```

### RoomCreatorPage.vue Script 部分

```javascript
import SimpleDependencyPage from './SimpleDependencyPage.vue'

// 删除 goToSimpleDependency 方法
```

### SimpleDependencyPage.vue 适配

需要修改简版页面以支持嵌入式显示：
- 添加 `embedded` prop
- 当 `embedded=true` 时，隐藏顶部导航栏（返回按钮等）
- 保持数据加载和保存逻辑

## 样式调整

添加简版内容区域的样式：
```css
.simple-dependency-content {
  height: calc(100vh - 160px);
  width: 100%;
  overflow: hidden;
}
```
